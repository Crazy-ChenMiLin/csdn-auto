# 我的私有自动化技能库搭建记 🤖

今天想分享一下我最近搭建的自动化技能库，用于学习自动化技术。

## 为什么要做这个？

平时工作中经常需要重复操作：
- 写文章要逐个平台发布
- 检查账号登录状态
- 整理学习内容

所以决定用 Playwright 搭建一套自动化工具，纯粹为了学习和技术研究。

## 目前有哪些技能？

### 1️⃣ CSDN 自动保存草稿
用 Node.js + Playwright 实现，可以自动登录并保存文章草稿。

### 2️⃣ 小红书自动发布
用 Python + Playwright 实现：
- 支持图文笔记发布
- 支持长文章发布
- 一键排版功能

### 3️⃣ 抖音自动发布
同样是 Python + Playwright：
- 文章导入发布
- 图文内容发布

## 技术栈选择

- **Playwright**: 浏览器自动化神器
- **Python/Node.js**: 根据平台特点选择
- **Cookie 登录**: 避免频繁扫码

## 学习收获

搭建过程中学到很多：
- 各平台的前端结构差异
- Cookie 管理和注入技巧
- 反自动化检测的应对
- 网络请求抓包分析

## 安全提醒 ⚠️

Cookie 文件相当于账号密码，一定要：
- 妥善保管，不要泄露
- 不要上传到公开仓库
- 使用私有仓库保存代码

这套工具仅供学习研究自动化技术使用，使用需遵守各平台的服务条款。
